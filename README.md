Railway Management System (SQL)

📌 Project Overview
This project is a Railway Management System developed using SQL. It is designed to manage train details, passenger information, and ticket booking operations efficiently.

🛠️ Technologies Used
- SQL (MySQL)

📂 Features
- Train management (Train ID, source, destination, timings)
- Passenger information storage
- Ticket booking and cancellation
- Seat availability tracking
- Data retrieval using SQL queries

🗄️ Database Design
The system includes the following tables:
- **Trains** (Train_ID, Train_Name, Source, Destination)
- **Passengers** (Passenger_ID, Name, Age)
- **Bookings** (Booking_ID, Train_ID, Passenger_ID, Seat_No)


🔍 SQL Concepts Used
- SELECT, INSERT, UPDATE, DELETE
- JOIN operations
- Primary Key & Foreign Key constraints
- Data normalization

▶️ How to Run
1. Open MySQL or any SQL database tool
2. Import the `.sql` file
3. Execute queries to create tables and manage data

🎯 Learning Outcome
- Understanding of database design
- Hands-on experience with SQL queries
- Implementation of relational database concepts
